## How to Run App
- Be sure your computer is already installed node.js and npm.
- Install the module with command : 
```bash 
  npm install
```
- Run developer mode by command :
```bash 
  npm start
```
- Build production mode by command :
```bash 
  npm run build
```
- Run analyze package by command :
```bash 
  npm run analyze
```
